# アーユルヴェーダ診断ページの作成

## 目標
- 以前作成した QuestionGame の応用
- フレームワークを使用せず、Vanilla JS で構築

## 使用言語
- HTML
- CSS
- JavaScript

## 使用技術

### HTML5
- 画面構成
- UIコンポーネントの設計

### CSS3
- レイアウト
- ボタンデザイン
- 画面遷移のスタイル

### JavaScript（Vanilla JS）
- DOM操作
- イベントハンドリング
- 状態管理（ゲーム進行・スコア管理）
- 配列データの管理
- 画面レンダリング

## 応用として行ったこと
- 親ページで作成したデータを別ページへ受け渡し
- 別ウィンドウへの遷移処理

## 改善点
- 質問を間違えると最初からやり直しになるため、戻れるようにしたい
- UI をより魅力的にしたい（画像追加・デザイン改善）
- 質問形式をラジオボタンに変更し、残りの質問数なども表示したい
